# MERN Live Chat Application - Backend

## 📌 Features
- REST APIs for chat and user management  
- User authentication using JWT  
- Real-time communication using Socket.io  
- MongoDB database integration  
- Image upload support using Cloudinary  

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- Socket.io  
- JWT Authentication  
- Cloudinary

## 🔗 Frontend Repository
https://github.com/radhanapatel33/mern-live-chat-app

## 📌 Note
This backend handles APIs, authentication, and real-time communication for the chat application.
